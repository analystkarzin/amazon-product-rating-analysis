# amazon-product-rating-analysis
Analysis of Amazon product ratings to identify top-rated products


## Business Question
Which products have the highest ratings on Amazon, and how many products have a perfect rating (5.0)?

## Dataset
- Source: Amazon product dataset, kaggle.com
- Total records: 1466
- Key columns: Product Id, Rating

## Tools Used
- Excel (data cleaning and filtering)
- GitHub (project documentation)

## Data Cleaning Steps
- Removed duplicate products
- Handled missing values
- Filtered products with rating = 5.0

## Analysis
- Identified all products with a 5.0 rating
- Counted and listed top-rated products

## Key Insights
- Several products achieved a perfect 5.0 rating
- Most top-rated products had fewer reviews, indicating possible bias
